# network-config-generator
多厂商网络设备开局配置生成工具，支持锐捷、华为、华三、思科，覆盖接入 / 核心交换机、路由器、AC 控制器。
## 一、项目基本信息
### 1.1 项目名称
Network-Config 网络设备配置脚本生成器（桌面单机版）
### 1.2 项目定位
专为网络工程师、运维人员打造的轻量化桌面工具，聚焦主流厂商网络设备配置脚本自动化生成，采用模块化解耦设计，操作简洁、运行稳定、无冗余功能，可直接打包为 Windows 单文件 EXE，无需安装、无需依赖外部环境，实现表单参数录入即可快速输出可直接部署的设备配置脚本。
### 1.3 项目核心原则
模块独立、配置分离、稳定兼容、简洁易用，杜绝代码重复、配置耦合，保障工具长期运行无卡顿、无异常。
## 二、核心业务模块规划
### 2.1 支持厂商（4 大类）
华为、锐捷、华三、思科
### 2.2 支持设备类型（4 大类）
接入交换机、核心交换机、路由器、无线控制器（AC）
### 2.3 模块架构
采用**4 厂商 ×4 设备**组合模式，共计**16 个独立业务模块**，每个模块对应独立操作页面、独立配置文件，单独开发、单独维护，互不干扰、不共用配置、不交叉调用。
### 2.4 核心功能
1. 主界面实现 16 个独立模块页面便捷切换，布局统一、操作一致
2. 各模块专属表单参数录入，支持基础网络、设备、协议参数填写
3. 读取对应独立配置文件模板，自动生成合规配置脚本
4. 脚本一键复制、表单快速清空、参数基础校验
5. 本地单机运行，无联网需求，数据本地处理
## 三、技术栈选型
Python 3 + PyQt5
