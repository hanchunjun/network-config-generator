## 1、单步独立指令

1. 项目架构全盘梳理

   @exploration 完整扫描当前项目，输出目录结构、核心文件、模块职责、调用关系及整体架构总结。
2. 代码规整精简（不改功能）

   @code-simplifier 对项目全部代码做精简优化、去除冗余、拆分长函数、统一格式，保留原有所有业务逻辑不变，输出优化清单。
3. 全项目代码风险审查

   @code-review 对整个项目代码做全面审查，排查逻辑漏洞、边界缺失、异常漏处理、隐性 Bug 与潜在风险，输出整改建议。
4. 自动生成全套测试用例

   @test-engineer 为本项目设计完整测试验证路径，覆盖正常、边界、异常、易遗漏场景，生成可直接运行的测试代码。
5. 报错精准定位修复

   @debugger 依据项目报错信息与运行日志，定位问题根因，给出可直接落地的修复方案。

## 2、常用组合指令

架构梳理 + 代码整容

@exploration 扫描项目架构 && @code-simplifier 全局代码规整优化，不改动业务功能

代码优化 + 风险体检

@code-simplifier 精简规整代码 && @code-review 全项目风险审查

## 3、全套终极流水线

@exploration 项目架构扫描 -> @code-simplifier 代码优化规整 -> @code-review 全量风险审查 -> @test-engineer 自动化测试全覆盖
