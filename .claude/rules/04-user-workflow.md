# 全流程用户操作步骤（V0.3.2）

## 标准操作流程

1. 双击 `NetworkConfigGenerator.exe` 启动（无需安装，首次自动创建目录）
2. 进入 **📁 新建项目**，创建项目，添加设备清单（支持粘贴导入）
3. 进入 **⚙ 模型设置**，配置AI模型 → 测试连通 → 保存（自动加密）
4. 进入 **🔍 单点运维**，添加单台设备 → 测试连接 → 执行巡检/备份/AI分析
5. 进入 **🔧 项目运维**，选择项目 → 执行批量任务 → 查看结果Tab → AI分析
6. 需要深度分析时，进入 **🤖 专家工作站**，选文件 → Agent快捷指令 → 编辑Prompt → 发送
7. **★V0.3.2** 批量生成配置脚本时，进入 **📜 命令生成**，编写模板(%a~%e占位) → 选择模式（固定重复/差异化生成）→ 配置参数(基数/步长) → 一键批量生成 → 复制/导出

> **试用模式说明**：未激活用户可使用「锐捷接入交换机配置」+「📜 命令生成」两个功能，其他功能需激活后使用。

## 三档AI分析路径

### 快速分析（第一层：嵌入式）
1. 在**单点运维**或**项目运维**中执行备份/巡检
2. 切换到对应Tab，选中文件
3. 点击 `[🔍AI合规巡检]` 或 `[🩺AI故障诊断]` → 自动执行 → 结果归档

### 深度分析（第二层：Agent快捷指令）
1. 进入**专家工作站**，选择文件
2. 点击 `[🔍一键合规巡检]` 或 `[🩺一键故障诊断]`
3. Prompt自动填入（可编辑修改）→ 点击 `[▶发送分析]`
4. 查看结果 → `[💾保存报告]` 或 `[📤导出TXT]`

### 自由对话（第三层）
1. 进入**专家工作站** → 💬自由对话Tab
2. 输入任意问题或粘贴内容 → 点击 `[▶发送分析]`
3. 支持多文件对比分析、多轮迭代提问

## 文件管理统一操作

所有文件列表区域：
- `[🔄刷新]` — 重新扫描目录，更新列表
- `[📂打开]` — 用系统默认程序打开选中文件
- `[🗑删除]` — 删除选中文件（有确认对话框）

## 新增安全特性

1. **安全确认**：导出文件、删除设备、查看密码时需要确认
2. **日志记录**：所有操作记录到 logs/netops_*.log
3. **密码加密**：设备密码自动加密存储（V0.2版本）
4. **原子操作**：文件保存使用原子操作，防止损坏

## 性能说明

1. **配置缓存**：相同配置快速生成（LRU缓存128条目）
2. **启动速度**：约3-5秒
3. **内存使用**：约150-200MB（运行时）
4. **EXE大小**：约48.7MB

## 首次使用要点

1. 先建项目再配模型，顺序可互换
2. AI模型配置完成自动加密为 `.enc` 文件
3. 模型设置中切换激活模型后，所有AI按钮立即生效
4. 删除操作均有确认对话框，防止误操作

## 数据迁移

- 旧版本密码自动迁移到V0.2格式
- 密钥文件 config/key_info.json 需妥善保管
- 拷贝到其他电脑需重新配置AI模型（密钥绑定机器ID）

## 注意事项

1. **安全警告**：导出文件包含明文密码，请妥善保管
2. **密钥备份**：config/key_info.json 是解密关键，建议备份
3. **日志管理**：logs 目录会积累日志，定期清理
4. **自动化仅限只读**：不自动修改设备配置，高危操作生成人工清单

### 📜 批量命令生成器使用步骤

#### 快速上手（使用预置模板）
1. 进入 **📜 命令生成** 页面（快捷键 `Ctrl+6`）
2. 点击「预置模板」下拉框，选择内置模板（如「批量接口配置」）
3. 查看左侧模板编辑区已加载的命令模板
4. 右侧参数配置区调整各参数的基数、步长
5. 如需循环生成，勾选「循环个数」并设置数量
6. 查看下方「生成结果」区域实时预览生成的命令列表
7. 点击「复制全部」或「导出TXT」保存结果

#### 自定义模板模式
1. 清空模板编辑区，编写自定义命令模板
2. 使用占位符 `%a`, `%b`, `%c`, `%d`, `%e` 表示可变部分
3. 示例模板：
   ```
   vlan %a
    name Sales_Floor_%b
   interface GigabitEthernet0/0/%a
    description Connect_to_Switch_%b
    switchport access vlan %a
   ```
4. 为每个使用的占位符配置参数：
   - `%a`: base=10, step=1, loop=8 → 生成 10,11,12...17
   - `%b`: base=1, step=1, loop=8 → 生成 1,2,3...8
5. 调整参数后自动重新生成

#### 高级参数配置
- **基数(base)**: 起始数值
- **步长(step)**: 每次递增/递减量（支持负数倒计数）
- **重复次数(repeat)**: 同一值重复输出多次
- **循环个数(loop)**: 参数循环多少个不同值
- 参数间取最长循环次数作为总生成行数

#### 实际应用场景示例

**场景1：批量创建VLAN**
```
模板：vlan %a
     name VLAN_%b
参数：%a: base=100, step=1, loop=10  →  VLAN 100~109
     %b: base=100, step=1, loop=10  →  名字 VLAN_100~VLAN_109
```

**场景2：批量配置接口IP**
```
模板：interface Vlan%a
      ip address 192.168.%a.1 255.255.255.0
      description Gateway_for_VLAN_%a
参数：%a: base=10, step=1, loop=5  →  Vlan10~Vlan14, 192.168.10.1~192.168.14.1
```

**场景3：生成ACL规则**
```
模板：access-list %a permit ip host 192.168.%b.%c any
参数：%a: base=100, step=0, repeat=50  →  ACL编号固定100
     %b: base=1, step=0, loop=25       →  第3段 1~25
     %c: base=1, step=1, loop=25       →  第4段 1~25
```
