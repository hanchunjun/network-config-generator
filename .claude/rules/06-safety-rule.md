# 安全执行铁律 & 成品核心能力总结（V2.1）

## 安全执行铁律

1. 自动化仅允许 show / display / ping / traceroute 只读指令，绝不自动修改设备配置
2. 接口启停、路由变更、设备重启、配置保存等高风险操作，禁止自动化，统一生成人工清单
3. AI密钥、设备账号密码仅本地存储，不上传云端
4. 多项目严格物理隔离，杜绝跨项目误操作、数据混淆
5. 设备敏感凭证仅留存项目本地，不对外泄露扩散

## 原有能力完整保留

- 四厂商四类设备可视化配置脚本生成
- 原生PyQt5界面简洁稳定
- 配置生成模块可独立运行不受新增功能影响

## V2.1 核心能力

1. 政企标准化多项目隔离管理 + 项目总览（卡片网格 + 状态灯 + 全局统计）
2. 全网批量备份、自动化巡检、故障二次核查
3. 单点设备全链条运维（测试连接/巡检/备份/AI诊断/AI审计）
4. AI能力三层架构：嵌入式一键分析 + Agent快捷指令 + 自由对话
5. **双层AI分析（V2.1新增）**：本地规则引擎预检 → 精准上下文提取 → AI精审，Token消耗降低70%+
6. 双层合规审计：本地规则引擎预检 + AI精审（全中文报告）
7. 全自定义大模型接口适配，模型切换自动生效
8. 文件管理三件套统一标准
9. 单EXE零依赖开箱即用（47.1MB）

## 核心安全优化

### 1. 密钥管理系统（V2版本）
- **动态派生：** 基于机器ID派生密钥，替代硬编码
- **版本控制：** 支持V1/V2格式，自动迁移
- **密钥备份：** 提供密钥备份和恢复机制

### 2. 加密算法增强
- **算法升级：** AES-GCM算法
- **多版本兼容：** 支持旧版本数据自动迁移

### 3. 输入验证和清理
- **IP地址验证：** 格式、范围、保留地址检查
- **设备数据验证：** 厂商、类型、参数合法性
- **命令注入防护：** 特殊字符过滤

### 4. 安全确认机制
- **导出确认：** 导出文件包含明文密码，需要确认
- **删除确认：** 删除设备或项目需要确认
- **密码查看：** 查看明文密码需要确认

### 5. 文件操作安全
- **原子操作：** 文件写入使用原子操作，防止损坏
- **数据完整性：** 确保文件写入完整

### 6. 日志和审计
- **操作日志：** 详细记录所有用户操作
- **错误日志：** 记录系统错误和异常

## 性能优化成果

- **配置缓存：** LRU缓存，128条目
- **VLAN批量：** 范围压缩优化
- **字符串处理：** join替代+=
- **表格操作：** 减少UI组件访问
