## 一、产品简介

本工具是一款面向政企网络的自动化运维桌面软件，免安装、双击直跑、整体拷贝即用。支持锐捷、华为、H3C、思科全系列网络设备，覆盖开局建站配置生成、日常批量运维、单点故障排查、AI智能诊断与合规审计全场景。

### 核心特点

- **单EXE绿色免安装**：约47MB，拷贝到任何Windows电脑直接双击运行
- **试用模式**：未激活可体验「锐捷接入交换机配置」功能，全功能需授权激活
- **多项目物理隔离**：每个项目数据独立存储，杜绝跨项目混淆
- **AI三层架构**：一键自动分析 → Agent快捷指令 → 自由对话，从简单到深度
- **双层AI分析**：本地规则引擎预检 + AI精审，异常精准定位，Token消耗降低70%+
- **全中文报告**：所有AI输出均为中文，含分级整改建议
- **安全第一**：自动化仅执行只读指令，绝不修改设备配置

## 二、快速上手（5分钟）

### 第一步：启动

双击 `NetworkConfigGenerator.exe`，首次启动会自动创建目录结构，约3-5秒进入主界面。

> **试用模式**：未激活用户进入主界面后，窗口标题显示 `[试用模式]`，仅可使用「锐捷接入交换机配置」功能。点击其他导航项会弹出激活提示，引导联系管理员激活。

### 第二步：激活软件（试用用户跳过）

1. 点击右上角「关于」→「软件激活」按钮
2. 复制本机机器码（32位大写字符串）
3. 将机器码发送给管理员【老韩】（QQ: 223518 / WeChat: tachlaohan）
4. 收到激活码后填入输入框，点击「立即激活」
5. 激活成功后全部功能自动开放

### 第三步：创建项目

1. 点击左侧导航 **📁 新建项目**
2. 点击 `＋ 新建项目` → 输入项目名称 → 确定
3. 点击 `编辑设备清单` → 按格式录入设备信息
4. 设备格式：`IP，厂商，设备类型，用户名，密码，协议，特权密码`
5. 支持从Excel粘贴导入

### 第四步：配置AI模型

1. 点击 **⚙ 模型设置**
2. 选择厂商 → 填写Base URL → 填写API Key → 填写模型名
3. 点击 `测试连接` 验证连通性
4. 点击 `保存配置`
5. 支持添加多套模型配置，激活哪套用哪套

### 第五步：开始使用

现在可以：
- 进 **🖥 设备配置** 生成四厂商配置脚本
- 进 **🔍 单点运维** 排查某台设备
- 进 **🔧 项目运维** 批量备份/巡检整个项目
- 进 **🤖 专家工作站** 进行深度AI分析

## 三、六大功能模块

### 📁 新建项目（项目总览）

**用途：** 管理所有项目，一目了然查看全局状态。

**操作：**
- `＋ 新建项目`：创建新项目，自动生成标准化目录
- `✏ 重命名`：修改项目名称
- `🗑 删除`：删除整个项目（有确认对话框）
- `📂 打开目录`：用文件管理器打开项目目录
- `编辑设备清单`：添加/修改/删除设备信息

**项目总览卡片：** 每个项目显示状态灯、设备数量、最近备份/巡检时间、故障设备数，点击「进入项目」一键切换。

**操作流程：**

**4个结果Tab：**

| Tab | 内容 | AI能力 |
|-----|------|--------|
| 💾 备份文件 | 备份文件列表 + 内容预览 | `[🔍AI合规巡检]` |

## 四、软件激活体系

### 激活方式

本软件采用设备绑定激活模式，一机一码，不可通用。

**激活流程：**
1. 点击右上角导航栏「🔓 未激活」按钮（或「关于」→「软件激活」）
2. 复制本机机器码（32位大写字符串，基于CPU+硬盘序列号生成）
3. 将机器码发送给管理员【老韩】
4. 收到16位激活码后填入，点击「立即激活」
5. 激活成功，全部功能开放，导航栏显示「✅ 永久激活」或「✅ 剩XXX天」

### 激活方案

| 方案 | 说明 | 适用场景 |
|------|------|---------|
| 方案A | 纯离线永久激活，无网络依赖 | 熟人内部、小范围分发 |
| 方案B | 离线使用 + 180天静默黑名单校验 | 对外分发、商用、需可控 |
| 方案C | 管理员独立制码工具 | 管理员统一出码、台账管理 |

### 有效期选项

管理员制码时可选择10档有效期：

| 选项 | 天数 | 适用场景 |
|------|------|---------|
| 永久 | 0 | 方案A纯离线，长期信任 |
| 5年 | 1825 | 长期授权 |
| 10年 | 3650 | 超长期授权 |
| 3年 | 1095 | 中长期授权 |
| 2年 | 730 | 中期授权 |
| 1年 | 365 | 年度授权 |
| 半年 | 180 | 方案B黑名单可控 |
| 季度 | 90 | 短期试用 |
| 月度 | 30 | 临时试用 |
| 周度 | 7 | 极速试用 |

### 剩余天数查看

激活后，主窗口导航栏右上角激活按钮实时显示授权状态：
- **✅ 永久激活**（绿色）：永久授权，无到期限制
- **✅ 剩XXX天**（绿色）：有期限授权，剩余天数 > 30天
- **✅ 剩XXX天**（橙色）：有期限授权，剩余天数 ≤ 30天（即将到期预警）
- **🔓 未激活**（红色）：未激活或已过期

点击已激活的按钮可弹出授权详情，查看激活时间、到期时间、剩余天数等信息。

### 硬件绑定说明

- 机器码由CPU序列号 + 硬盘物理序列号生成，重装系统/更新驱动不改变
- 更换CPU或硬盘将导致机器码变更，需重新申请激活
- 激活文件加密存储在 `activation/license.dat`
- 有期限授权到期后自动失效，需联系管理员重新激活

### 管理员制码工具

管理员使用独立的 `AdminKeyGenTool.exe`（约41MB）：
- 粘贴用户机器码 → 一键生成16位激活码
- 10档有效期可选（永久~7天）
- 自动记录授权台账（AES-GCM加密）
- 支持台账备份/恢复/导出
- 支持黑名单管理（方案B）
- 窗口启动自动适配屏幕80%大小并居中

## 五、AI能力速查

### 三层架构

```
第一层：嵌入式按钮 → 单点运维/项目运维 → 一键自动执行
第二层：Agent快捷指令 → 专家工作站 → 可编辑Prompt
第三层：自由对话 → 专家工作站 → 完全自定义
```

### 双层AI分析

每次 AI 分析均经历两层处理，确保精准且节省 Token：

```
Layer 1：本地规则引擎（确定性规则，毫秒级）
  → 从巡检报告/配置中提取结构化异常
  → 生成精简异常摘要 + 精准上下文片段
         ↓
Layer 2：AI 大模型（语义分析，深度推理）
  → 仅接收精简摘要 + 异常周边上下文（而非全文）
  → 深度根因分析 + 补充遗漏 + OSI分层排查命令
```

**Token 优化效果**：
- 合规审计：从 18.4KB → 5.0KB（压缩 73%）
- 故障诊断：从 151.6KB → 5.5KB（压缩 96%）

## 七、数据安全

- AI配置加密存储，采用AES-GCM算法
- 密钥绑定机器ID，拷贝到其他电脑需重新配置模型
- 所有操作记录到 `logs/` 目录
- 删除/导出等危险操作有确认对话框
- 批量运维仅执行只读指令（show/display/ping）
- 高危操作（接口启停/路由变更/配置保存）仅生成人工清单
- 激活授权文件AES-GCM加密存储
- 管理员台账AES-GCM加密存储，支持备份恢复

## 八、目录结构

程序运行后，EXE所在目录自动生成以下结构：

```
程序根目录/
├── NetworkConfigGenerator.exe          ← 主程序
├── AdminKeyGenTool.exe                 ← 管理员制码工具
├── config/                             ← 系统配置（密钥/项目索引/AI配置）
├── activation/                         ← 用户端激活（授权文件/黑名单校验）
├── admin_data/                         ← 管理员数据（加密台账/黑名单/备份）
├── logs/                               ← 运行日志
├── single/                             ← 单点运维数据（隔离）
└── projects/                           ← 项目数据（多项目隔离）
    └── 01_项目名称/
        ├── config/device_list.txt      ← 设备清单
        ├── config_backup/              ← 配置备份
        ├── output/                     ← 批量运维输出
        └── report/                     ← 报告归档
```

## 九、常见问题

**Q：启动后进入试用模式，标题显示 [试用模式]？**
A：正常现象。未激活用户可体验「锐捷接入交换机配置」功能，点击其他功能会弹出激活提示。如需全功能，请联系管理员【老韩】（QQ: 223518 / WeChat: tachlaohan）激活。

**Q：如何激活软件？**
A：点击导航栏右上角「🔓 未激活」按钮（或「关于」→「软件激活」）→ 复制机器码 → 发送给管理员 → 填入激活码 → 点击激活。

**Q：激活后如何查看剩余天数？**
A：导航栏激活按钮直接显示：永久激活显示「✅ 永久激活」，有期限显示「✅ 剩XXX天」。点击按钮可查看完整授权详情（激活时间、到期时间、剩余天数）。橙色表示剩余≤30天，即将到期。

**Q：激活码无效怎么办？**
A：请检查：① 激活码是否为16位大写字母+数字；② 激活码是否与当前设备匹配；③ 联系管理员【老韩】确认。

**Q：更换电脑后激活失效？**
A：正常现象。激活绑定硬件（CPU+硬盘），更换电脑需重新提交机器码申请激活。

**Q：AI按钮点不了，提示"未配置模型"？**
A：进入「模型设置」，确认已保存至少一套模型配置并设为激活状态。

**Q：AI分析结果不理想？**
A：进入「专家工作站」，选择对应Agent快捷指令后可以手动编辑Prompt优化问题描述。

**Q：拷贝到其他电脑AI配置丢失？**
A：正常现象，AI密钥绑定机器ID，需在新电脑上重新配置模型。

**Q：设备连接失败？**
A：先用「单点运维」的 `🧪 测试连接` 功能验证Ping和SSH/Telnet是否能通。

**Q：项目找不到？**
A：检查 EXE 同级目录下 `projects_config.json` 和 `projects/` 目录是否存在。

**Q：提示"授权已失效"？**
A：设备被列入黑名单（方案B），请联系管理员【老韩】重新审核办理。
