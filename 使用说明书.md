## 一、产品简介

本工具是一款面向政企网络的自动化运维桌面软件，免安装、双击直跑、整体拷贝即用。支持锐捷、华为、H3C、思科全系列网络设备，覆盖开局建站配置生成、日常批量运维、单点故障排查、AI智能诊断与合规审计全场景。

### 核心特点

- **单EXE绿色免安装**：约48.7MB，拷贝到任何Windows电脑直接双击运行
- **登录认证**：程序启动强制登录，默认账户 admin/admin，支持自主修改用户名和密码
- **试用模式**：未激活可体验「锐捷接入交换机配置」和「命令生成」功能，全功能需授权激活
- **多项目物理隔离**：每个项目数据独立存储，杜绝跨项目混淆
- **AI三层架构**：一键自动分析 → Agent快捷指令 → 自由对话，从简单到深度
- **双层AI分析**：本地规则引擎预检 + AI精审，异常精准定位，Token消耗降低70%+
- **全中文报告**：所有AI输出均为中文，含分级整改建议
- **安全第一**：自动化仅执行只读指令，绝不修改设备配置
- **三主题切换**：VS Code（深色技术感）、Raycast（紫橙渐变毛玻璃）、Business（浅色政企风），切换即时生效；Windows 标题栏跟随主题深浅变化；输入框边框在深色主题下自动提亮
- **设备绑定激活**：一机一码，支持10档有效期（永久~7天）
- **密码加密存储**：AES-256-GCM加密，杜绝明文泄露

## 二、快速上手（5分钟）

### 第一步：启动与登录

双击 `NetOps.exe`，程序启动后依次执行：

1. **激活校验**：自动检查本机激活状态
2. **登录认证**：弹出登录窗口，输入用户名和密码
   - 首次运行默认账户：`admin` / `admin`
   - 登录成功后进入主界面
   - 登录窗口无关闭按钮，必须正确登录才能进入

> **试用模式**：未激活用户通过登录后进入主界面，窗口标题显示 `[试用模式]`，仅可使用「锐捷接入交换机配置」和「命令生成」功能。点击其他导航项会弹出激活提示。试用模式下打开激活弹窗后，可点击「稍后再说」返回试用主窗口。

### 第二步：激活软件

1. 点击右上角导航栏「🔓 未激活」按钮（或「关于」→「软件激活」）
2. 复制本机机器码（32位大写字符串）
3. 将机器码发送给管理员【老韩】（QQ: 223518 / WeChat: tachlaohan）
4. 收到激活码后填入输入框（16位或18位），点击「立即激活」
5. 激活成功后全部功能自动开放

### 第三步：修改默认密码（安全建议）

首次登录后，强烈建议立即修改默认密码：

1. 点击导航栏右上角「账户管理」按钮
2. 输入新用户名（可选，默认为 admin）
3. 输入新密码（≥8位，必须同时包含大写字母、小写字母、数字）
4. 再次输入确认密码
5. 点击「保存」，提示成功后重启软件生效

### 第四步：创建项目

1. 点击左侧导航 **📁 新建项目**
2. 点击 `＋ 新建项目` → 输入项目名称 → 确定
3. 点击 `编辑设备清单` → 按格式录入设备信息
4. 设备格式：`IP，厂商，设备类型，用户名，密码，协议，特权密码`
5. 支持从Excel粘贴导入

### 第五步：配置AI模型

1. 点击 **⚙ 模型设置**
2. 选择厂商 → 填写Base URL → 填写API Key → 填写模型名
3. 点击 `测试连接` 验证连通性
4. 点击 `保存配置`
5. 支持添加多套模型配置，激活哪套用哪套

### 第六步：开始使用

现在可以：
- 进 **🖥 设备配置** 生成四厂商配置脚本
- 进 **🔍 单点运维** 排查某台设备
- 进 **🔧 项目运维** 批量备份/巡检整个项目
- 进 **🤖 专家工作站** 进行深度AI分析

## 三、登录认证与账户管理

### 登录认证

程序启动时（激活校验通过后）强制弹出登录窗口，身份核验通过后方可进入主界面。

**登录窗口特性：**
- 无关闭按钮，禁止ESC，必须正确登录才能进入
- 首次运行自动生成默认账户 `admin` / `admin`
- 登录失败提示「用户名或密码不正确，请重新输入」，清空密码框
- 连续登录失败不锁定（防止恶意尝试），但每次失败均有明确提示

### 账户管理

导航栏「关于」按钮左侧设有「账户管理」入口，点击弹出账户管理窗口。

**修改账户操作：**
1. 点击导航栏「账户管理」按钮
2. 输入新用户名（留空则保持原用户名）
3. 输入新密码（≥8位，必须同时包含大写字母、小写字母、数字）
4. 再次输入确认密码（两次必须一致）
5. 点击「保存」

**密码复杂度要求：**
- 长度 ≥ 8 位
- 必须同时包含大写字母（A-Z）、小写字母（a-z）、阿拉伯数字（0-9）
- 示例合规密码：`Admin1234`、`MyP@ssw0rd`

**安全机制：**
- 密码采用 AES-256-GCM 加密存储于 `config/account.json`
- 加密密钥与机器绑定，拷贝账户文件到其他电脑无法直接使用
- 账户文件损坏时自动重置为默认账户 admin/admin

## 五、软件激活体系

### 激活方式

本软件采用设备绑定激活模式，一机一码，不可通用。

**激活流程：**
1. 点击右上角导航栏「🔓 未激活」按钮（或「关于」→「软件激活」）
2. 复制本机机器码（32位大写字符串，基于CPU+硬盘序列号生成）
3. 将机器码发送给管理员【老韩】
4. 收到激活码后填入（16位旧格式或18位新格式），点击「立即激活」
5. 激活成功，全部功能开放，导航栏显示「✅ 永久激活」或「✅ 剩XXX天」

**激活码格式说明：**
- **16位**：旧格式激活码，永久授权（如 `F1637109D2A857DE`）
- **18位**：新格式激活码，含有效期编码（如 `F1637109D2A857DE05`，末尾 `05` 表示1年有效期）
- 两种格式均有效，16位码始终视为永久授权

### 激活方案

| 方案 | 说明 | 适用场景 |
|------|------|---------|
| 方案A | 纯离线永久激活，无网络依赖 | 熟人内部、小范围分发 |

| 方案C | 管理员独立制码工具 | 管理员统一出码、台账管理 |

- **✅ 剩XXX天**（绿色）：有期限授权，剩余天数 > 30天
- **✅ 剩XXX天**（橙色）：有期限授权，剩余天数 ≤ 30天（即将到期预警）
- **🔓 未激活**（红色）：未激活或已过期

点击已激活的按钮可弹出授权详情，查看激活时间、到期时间、剩余天数等信息。

### 硬件绑定说明

- 机器码由CPU序列号 + 硬盘物理序列号生成，重装系统/更新驱动不改变

- 激活文件加密存储在 `activation/license.dat`
- 有期限授权到期后自动失效，需联系管理员重新激活

## 八、数据安全

- 密钥绑定机器ID，拷贝到其他电脑需重新配置模型
- 所有操作记录到 `logs/` 目录
- 删除/导出等危险操作有确认对话框
- 批量运维仅执行只读指令（show/display/ping）
- 高危操作（接口启停/路由变更/配置保存）仅生成人工清单
- 激活授权文件AES-GCM加密存储
- 管理员台账AES-GCM加密存储，支持备份恢复
- 账户密码AES-GCM加密存储，杜绝明文泄露

## 十、常见问题

**Q：启动后弹出登录窗口，默认账户是什么？**
A：首次运行默认账户为 `admin` / `admin`。首次登录后建议立即通过「账户管理」修改密码。

**Q：登录密码输错了怎么办？**
A：登录失败会提示「用户名或密码不正确，请重新输入」，清空密码框后可重新输入。登录窗口无法关闭，必须正确登录才能进入主界面。

**Q：忘记密码了怎么办？**
A：如果忘记密码，可删除 EXE 同级的 `config/account.json` 文件，程序下次启动将自动重置为默认账户 admin/admin。

**Q：如何修改密码？**
A：点击导航栏右上角「账户管理」按钮 → 输入新用户名和新密码 → 确认密码 → 保存。新密码必须≥8位，且同时包含大写字母、小写字母、数字。

**Q：启动后进入试用模式，标题显示 [试用模式]？**
A：正常现象。未激活用户通过登录后进入主界面，仅可使用「锐捷接入交换机配置」和「命令生成」功能，点击其他功能会弹出激活提示。如需全功能，请联系管理员【老韩】（QQ: 223518 / WeChat: tachlaohan）激活。

**Q：如何激活软件？**
A：登录后点击导航栏右上角「🔓 未激活」按钮（或「关于」→「软件激活」）→ 复制机器码 → 发送给管理员 → 填入激活码（16位或18位）→ 点击激活。

**Q：激活码无效怎么办？**
A：请检查：① 激活码是否为16位或18位大写字母+数字；② 激活码是否与当前设备匹配；③ 联系管理员【老韩】确认。

**Q：激活后如何查看剩余天数？**
A：导航栏激活按钮直接显示：永久激活显示「✅ 永久激活」，有期限显示「✅ 剩XXX天」。点击按钮可查看完整授权详情（激活时间、到期时间、剩余天数）。橙色表示剩余≤30天，即将到期。

**Q：试用模式下打开激活弹窗后无法关闭？**
A：试用模式下激活弹窗提供了「稍后再说」按钮，点击即可关闭弹窗返回试用主窗口。

**Q：更换电脑后激活失效？**
A：正常现象。激活绑定硬件（CPU+硬盘），更换电脑需重新提交机器码申请激活。

**Q：AI按钮点不了，提示"未配置模型"？**
A：进入「模型设置」，确认已保存至少一套模型配置并设为激活状态。

**Q：AI分析结果不理想？**
A：进入「专家工作站」，选择对应Agent快捷指令后可以手动编辑Prompt优化问题描述。

**Q：拷贝到其他电脑AI配置丢失？**
A：正常现象，AI密钥绑定机器ID，需在新电脑上重新配置模型。

**Q：设备连接失败？**
A：先用「单点运维」的 `🧪 测试连接` 功能验证Ping和SSH/Telnet是否能通。

**Q：项目找不到？**
A：检查 EXE 同级目录下 `projects_config.json` 和 `projects/` 目录是否存在。

**Q：提示"授权已失效"？**
A：设备被列入黑名单（方案B），请联系管理员【老韩】重新审核办理。
